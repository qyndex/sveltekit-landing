# SvelteKit Landing

Marketing landing page with hero, features, and pricing sections built with SvelteKit 2 and Svelte 5 runes.

**Stack:** Sveltekit | **Complexity:** simple
**Tags:** `sveltekit`, `svelte`, `landing`, `marketing`, `hero`, `pricing`, `typescript`

## Quick Start

```bash
npm install
npm run dev
```

## Docker

```bash
docker compose up --build
```
